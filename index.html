How to Download and Install

1. Click on the download button for your platform ( Windows or Android ).

2. For Windows : Unzip the downloaded file and run the 'Jet Striker.exe' file.

3. For Android : Install the APK file and follow on-screen instructions.

Gameplay Overview

Take command of your jets in this thrilling 2D shooter! On PC, use WASD or Arrow Keys to dodge enemy fire and navigate the battlefield. On Android, intuitive touch controls let you maneuver with precision. Engage in intense dogfights, destroy enemies, and survive the storm of bullets in this fast-paced aerial combat adventure.
